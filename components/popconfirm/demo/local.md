#### 国际化
使用 `okText` 和 `cancelText` 自定义按钮文字。
</cn>

<us>
#### Locale text
Set `okText` and `cancelText` props to customise the button's labels.
</us>

```html
<template>
  <a-popconfirm title="Are you sure？" okText="Yes" cancelText="No">
    <a href="#">Delete</a>
  </a-popconfirm>
</template>
```
